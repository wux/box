About Golden Boy Boxing

Golden boy boxing is a boxing website created for people that has a passion for boxing. This website features some of the famous boxers around the world. You can also check upcoming professional boxing compeition schedule,current boxing champions and boxing gym rankings.This website is made using XML(Extensible-Markup Language), XSLT and HTML, XML schema was also used to validate the XML file.

About The Website Creater

This xml website is created by Jessie, a student from Loughborough Univeristy studying Information Management and Computing Degree. Jessie volunteered at the London Olympic boxing event during 2012, during this period jessie met some famous boxers such as Lennox Lewis, Amir Khan which gave her motivational advice.this experience made her interested into boxing.
